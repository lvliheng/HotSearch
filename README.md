So Hot--Android热搜整合应用
=

### 功能：

现有三个模块：
* 微博--显示搜索结果的第一条内容，视频、图片可点击全屏查看。
* 抖音--列表中根据滚动位置自动播放，也可以点击视频全屏播放，并使用手势上下滑动翻页。
* V2EX--详情页面可使用手势左右滑动切换帖子。
(所有内容都可以长按标题在浏览器中打开)
